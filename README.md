# Cognifyz Technologies – Data Analysis Internship

## Project Overview
This project is part of the Data Analysis Internship at Cognifyz Technologies.  
The objective of this project is to analyze a restaurant dataset and extract meaningful insights using data analysis and visualization techniques.

## Tools & Technologies Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Tasks Completed

### Level 1
- Task 1: Data Overview
- Task 2: City Analysis
- Task 3: Price Range Distribution
- Task 4: Online Delivery Analysis
- Task 5: Restaurant Ratings

### Level 2
- Task 1: Restaurant Ratings Analysis
- Task 2: Cuisine Combination Analysis
- Task 3: Geographic Analysis
- Task 4: Restaurant Chains Analysis

## Key Insights
- Most restaurants fall under affordable price ranges.
- Restaurants offering online delivery tend to have higher ratings.
- Urban areas show dense restaurant clusters.
- Popular cuisine combinations dominate the dataset, while niche cuisines often receive higher ratings.
- Some restaurant chains balance both popularity and high customer satisfaction.
